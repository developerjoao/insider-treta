# Utilização

Ao abir o sistema, escolha um arquivo CSV de usuários para ser analisádo.

Depois, escolha uma data inicial e uma data final em que os dados vão ser analisados.

Escolha os arquivos CSV de Device, Http e Logon que vão ser processados.

Caso queira informações específicas sobre um usuário, digite o Id do usuário pós processamento, e clique no botão mostrar perfil.(Não conseguimos implementar totalmente.)

Para retornar quais usuários são suspeitos, clique em detectar anomalias.(Não conseguimos implementar totalmente.)
